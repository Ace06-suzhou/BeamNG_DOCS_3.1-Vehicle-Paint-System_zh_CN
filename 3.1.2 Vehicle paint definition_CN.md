车辆涂装本质上就是一组 [颜色调色板](/modding/vehicle/vehicle_paints/paint_materials_intro) 设置的预设。
它可以在生成车辆时通过车辆选择器挑选，也可以在车辆配置器的涂装菜单中重新喷漆时选择。

![](paints_veh.png)
*车辆选择器里展示的车辆涂装。这些就是车辆出厂时可供选择的原厂涂装颜色。*

![](paints_part.png)
*车辆配置器里展示的车辆涂装。除了出厂原装涂装，这个菜单还会列出合适的自定义涂装，以及用户自行保存的涂装预设。*

### Defining the paints

最简单的涂装定义方式是在车辆的 `info.json` 主文件中完成。该文件中每种涂装都以其名称作为标题，并包含以下几个基本属性：
- `baseColor`：涂装的基础 RGBA 颜色，每个分量取值范围为 0 到 1。其中的 Alpha 通道用于设置 `Chrominess` 参数，这是一个旧版遗留特性，在基于物理渲染系统中已不再支持。
- `clearCoat`: 定义清漆效果的强度大小。
- `clearCoatRoughness`: 定义清漆效果的粗糙程度。
- `metallic`: 定义金属度的大小。
- `roughness`: 定义粗糙度的大小。

```js
"paints":[
  "Mica": {
    "baseColor": [0.5,  0.42,  0.39,  1.2],
    "clearcoat": 0,
    "clearcoatRoughness": 0,
    "metallic": 0,
    "roughness": 0.07
  },
(...)
],
```
*基本的涂装定义*

涂装的另一个属性是 `class`。  当设置为 `custom` 时，该涂装会在车辆配置器的涂装菜单中出现在Custom列表下。
```js {hl_lines=[2]}
"Noce": {
  "class": "custom",
  "baseColor": [0.16,  0.1,  0.06,  1.2],
  "clearcoat": 0,
  "clearcoatRoughness": 0,
  "metallic": 0,
  "roughness": 0.07
},
```
*自定义涂装定义*

### 默认涂装

每种车辆配置都有自己独立的info.json 文件，命名规则为 `info_[配置名称].json`。比如配置名称是“standard”，对应的文件就是info_standard.json。在这个文件中，可以指定该配置的默认涂装名称。这个涂装会作为缩略图显示的涂装，也是当用户在车辆选择器中不选任何涂装时，车辆生成时默认使用的涂装。默认涂装会固定显示在涂装选器的第一个槽位，也就是左上角的位置。
![](default_paint.png)
*选中的车辆配置，默认涂装会在涂装选择器的第一个槽位中高亮显示。*

这个默认涂装通过 `defaultPaintName1` 属性进行设置。类似地，`defaultPaintName2` 和 `defaultPaintName3` 可用于为另外两个调色板指定默认涂装配置。
```js
"defaultPaintName1": "Yellow",
"defaultPaintName2": "Yellow",
"defaultPaintName3": "Black",
```
*默认涂装的配置*

车辆的主 `info.json` 文件也可以包含一份默认涂装列表，为每个调色板分别指定默认涂装。所选配置完全没有定义任何涂装时，就会使用这些默认值。因为只是作为备用方案，所以有时只定义第一个默认涂装，这时三个调色板就会全部使用同一个颜色。

内部处理时，`defaultPaintName` 属性的设置会被转换为 [多涂装设置](/modding/vehicle/vehicle_paints/multi_paint_setup)，相关细节将在后面的页面进一步说明。

### 用户配置中的涂装

用户的自定义配置不会拥有独立的 `info.json` 文件。涂装相关设置会保存在对应配置的 `.pc` 文件里面。三个调色板的设置都使用与基础涂装相同的格式，只是没有单独的名称。

```js
"paints":[
  {
    "baseColor":[
      0.86,
      0.66,
      0.1,
      1.2
    ],
    "clearcoat":1,
    "clearcoatRoughness":0.04,
    "metallic":0,
    "roughness":1
  },
  {
    "baseColor":[
      0.86,
      0.66,
      0.1,
      1.2
    ],
    "clearcoat":1,
    "clearcoatRoughness":0.04,
    "metallic":0,
    "roughness":1
  },
  {
    "baseColor":[
      0.83,
      0.83,
      0.83,
      1.2
    ],
    "clearcoat":1,
    "clearcoatRoughness":0.04,
    "metallic":0.3,
    "roughness":0.6
  }
],
```
*用户配置文件中的涂装设置*
