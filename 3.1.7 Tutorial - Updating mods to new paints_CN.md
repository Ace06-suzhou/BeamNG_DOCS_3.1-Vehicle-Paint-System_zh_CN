本教程将逐步说明如何将模组更新为使用新涂装系统。

{{< notice "warning" >}}
请务必仔细注意语法，特别是括号的使用方式！
{{< /notice >}}

### Library paints

如果你的模组里直接复制了原版车辆的某些涂装颜色，建议你先去默认的 [涂装库](/modding/vehicle/vehicle_paints/paint_library) 确认这些涂装是否已经存在，然后把你文件里的条目改成直接引用库里的涂装。
这样一来，如果将来原版涂装因为图形升级或语法调整而更新，你的模组也能自动同步这些变化。

```js
"paints": {
  "Night Blue": {
    "baseColor": [0.04,  0.06,  0.16,  1.0],
    "clearcoat": 0.8,
    "clearcoatRoughness": 0.07,
    "metallic": 0.8,
    "roughness": 0.5
  },
  (...)
},
```
*以前*

```js
"libraryPaints": [
  "Night Blue",
  (...)
],
```
*以后*


### 涂装分类

要减少车辆选择器里的涂装选项数量，可以把那些非出厂涂装移入 Custom 分类。

```js
"paints": {
  "Night Blue": {
    "baseColor": [0.04,  0.06,  0.16,  1.0],
    "clearcoat": 0.8,
    "clearcoatRoughness": 0.07,
    "metallic": 0.8,
    "roughness": 0.5
  },
  (...)
},
```
*以前*

```js {hl_lines=[3]}
"paints": {
  "Night Blue": {
    "class": "custom",
    "baseColor": [0.04,  0.06,  0.16,  1.0],
    "clearcoat": 0.8,
    "clearcoatRoughness": 0.07,
    "metallic": 0.8,
    "roughness": 0.5
  },
  (...)
},
```
*以后*

同样的方法也可以应用到 [涂装库](/modding/vehicle/vehicle_paints/paint_library) 中的部分涂装上。
```js
"libraryPaints": [
  "matte_black",
  (...)
],
```
*以前*

```js
"libraryPaints": [
  { "id": "matte_black",    "class": "custom" },
  (...)
],
```
*以后*


### 多涂装配置

在配置的info.json文件里，当涉及多个涂装颜色同时使用时（比如双色调皮肤），请将 `defaultPaintName1`、`defaultPaintName2` 和 `defaultPaintName3` 替换成 [多涂装设置](/modding/vehicle/vehicle_paints/multi_paint_setup)。
这样做能确保涂装组合在选择时颜色搭配合理且正确显示。

对于原厂配置，需要在车辆的主info.json文件中添加多涂装设置及其对应的涂装：
```js
"multiPaintSetups": [
  {
    "name": "Cream Two-Tone",
    "class": "factory",
    "paint1": "cream",
    "paint2": "light_brown",
    "paint3": "cream"
  },
  (...)
],
```

然后在对应配置的 info.json 文件中引用这些多涂装设置：
```js
"defaultPaintName1": "cream",
"defaultPaintName2": "light_brown",
"defaultPaintName3": "cream"
```
*以前*

```js
"defaultMultiPaintSetup":"Cream Two-Tone"
```
*以后*

对于自定义配置，最好直接在该配置的 info.json 文件中声明多涂装设置。  
这样它就只适用于这个特定配置。
```js
"defaultPaintName1": "matte_dune",
"defaultPaintName2": "army_green",
"defaultPaintName3": "matte_black"
```
*以前*

```js
"defaultMultiPaintSetup":
{
  "name": "Two-Tone Adventure",
  "class": "custom",
  "paint1": "matte_dune",
  "paint2": "army_green",
  "paint3": "matte_black"
}
```
*以后*
